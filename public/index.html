Welcome to the floating market! This website is a boilerplate for creating online market-places using Google Firebase and Stripe, specifically Stripe Standard Accounts using Stripe Connect . You'll need to create your own Firebase Project, get your own API keys, and add them as environmental variables for Firebase . Then it should work with just a firebase deploy command!
